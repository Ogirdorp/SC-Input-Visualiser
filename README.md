# SC-Input-Visualiser
A simple tool for pilot training. Python with the help of Claude.ai

Features

    Real-time stick X/Y visualization
    Throttle and reverse pedal bars
    Boost button indicator
    "Holding too long" warnings - alerts when holding a direction for >5 seconds
    Saves position and settings automatically
    No external dependencies - pure Python

Controls

    Drag anywhere to move the window
    Escape to quit
    Click Config to change device IDs
